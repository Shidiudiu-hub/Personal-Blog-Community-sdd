# Personal-Blog-Community-sdd

## 项目简介
个人博客社区系统是一个前后端分离的PC端Web应用，为用户提供文章发布、阅读、互动交流的一站式社区平台。

## 主要功能

### 📝 文章管理
- 支持 Markdown 编辑器撰写与实时预览
- 文章发布、编辑、删除
- 文章分类与标签管理
- 文章列表支持分类、标签、关键词搜索与多维度排序
- 文章详情页展示内容、作者信息、阅读量、点赞收藏数等

### 🏷️ 分类与标签
- 多级文章分类管理
- 标签云展示与热门标签推荐
- 文章与分类、标签的关联管理

### 💬 评论与回复
- 支持对文章发表评论
- 多级嵌套回复，形成树形评论结构
- 评论点赞与数量统计
- 评论编辑与删除

### ❤️ 点赞与收藏
- 文章点赞/取消点赞
- 文章收藏/取消收藏
- 个人点赞与收藏列表查看

### 🏠 用户中心
- 个人信息修改（头像、昵称、简介、密码等）
- 个人文章管理
- 个人数据查看

### 📊 数据统计
- 文章总数、热门文章统计
- 评论、点赞、收藏等互动数据统计

### 🔐 权限管理
- 登录用户可发布内容与互动
- 用户可管理自己的文章与评论
